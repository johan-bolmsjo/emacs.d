* Example .dir-locals.el file

Example ~.dir-locals.el~ file that overrides the C mode settings for a directory
and its child directories.

#+begin_src elisp
  ((c-mode . ((c-file-style . "linux")
              (indent-tabs-mode . t)
              (show-trailing-whitespace . t)
              (c-basic-offset . 8)
              (tab-width . 8)
              )))
#+end_src
